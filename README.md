# Google Forms Using OOP (C++)

This repository contains an implementation of a **Google Forms-style application using Object-Oriented Programming (OOP) in C++**. The project models core functionalities of a form system using classes, encapsulation, and other OOP principles to simulate form creation, question handling, and response collection.
